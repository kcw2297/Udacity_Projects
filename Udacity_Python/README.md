#### [과제 목표]

- 데이터를 올바른 정보로 보여줄 모델 생성 - models..py
- csv와 json파일에서 데이터 추출 - extract와 database..py
- 데이터 쿼리를 하고 필터링 - filters..py
- 데이터를 파일로 저장 - write..py
